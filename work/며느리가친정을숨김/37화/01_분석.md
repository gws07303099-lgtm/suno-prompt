# 37화 대본 분석

> 씬 번호는 대본 `#N` 그대로 사용(회차 연속번호). 총 3씬(#69–#71).

## #69. 시댁. 현관 — 저녁

- **장소/시간**: 시댁 현관 / 저녁
- **등장인물**: 도원(절뚝이며 귀가), 정숙(냉담한 시어머니)
- **갈등**: 표면 — 짧은 지시("방에 가서 쉬어라"). 이면 — 도원이 방으로 사라지자마자 혼잣말로 철저히 비하. 도원은 모름.
- **감정선**: 정숙의 이중성(점잖은 지시 ↔ 속 비하). 관객만 정숙의 본심을 목격하는 구조. 빌런 캐릭터 재각인.
- **전환점**: "쟤는 일생에 도움이 안돼" 혼잣말 — 박정숙 빌런 캐릭터화 완성 후 씬 컷.
- **페이싱**: 매우 짧고 단호. 빌런 각인 후 즉시 전환.
- **음악 시사**: 빌런(박정숙) 테마 삽입 지점. 정숙 혼잣말 시작~씬 끝까지. 카타르시스 없이 답답함만 남기는 컷.

## #70. 시댁. 지수부부의 방 — 저녁

- **장소/시간**: 시댁 지수부부 방 / 저녁
- **등장인물**: 도원 (혼자)
- **갈등**: 내면 — 기억 상실 상태에서 지수가 '보험금을 위해 남편을 죽이려 한 여자'임을 인지. 그러나 반지를 보는 순간 프로포즈 기억이 순간 스쳐 혼란. 기억이 돌아오려다 닫힘.
- **감정선**:
  1. **(전반 · 의혹)** 웨딩사진 보며 "무서운 여자랑 결혼했네" 독백 — 차갑고 혼란한 의혹.
  2. **(전환) 반지 발견 → 두통 → 플래시백** — 기억의 문이 순간 열렸다 닫히는 전환점.
  3. **(플래시백)** 한강공원 프로포즈 — 둘의 순수하고 따뜻한 부부애 원형. 경박한 유머(반지 꽉 끼는 도원)와 당황하는 지수로 행복감 표현.
  4. **(후반 · 상실)** 현재 복귀 → 기억 포기 → 한숨 → "우리는 어떤 부부였을까?" — 공허한 그리움.
- **전환점 1**: 반지 발견 + 두통 = 플래시백 트리거.
- **전환점 2**: "현재/" 복귀 = 기억 회복 실패 확인.
- **페이싱**: 차분한 의혹 구간 → 순간 전환(두통) → 따뜻한 플래시백 → 다시 정적 / 공허한 마무리. 감정 낙차 설계: 따뜻함(플래시백) → 상실(현재).
- **음악 시사**:
  - 전반(의혹 구간): 서스펜스/미스터리 언더스코어. 두통 직전 긴장 최고조 후 컷.
  - 플래시백: 따뜻한 부부애 테마 (상실의 그림자 포함 — 행복한 기억인 동시에 지금은 잃어버린 것).
  - 후반(기억 포기 ~ 한숨): 서러움·미니멀 이모셔널. 여백 극대화.

## #71. 시댁 앞. 거리 — 밤

- **장소/시간**: 시댁 앞 거리 (차 안) / 밤
- **등장인물**: 지수 (혼자, 차 안)
- **갈등**: 도원이 시댁 안에 있고 자신은 접근 불가. 그리움과 기다림.
- **감정선**: "돌아와줘, 도원 씨.. 기다리고 있을게.. 사랑해.." — 사랑과 그리움의 직접 고백. 훌쩍임 + F.O으로 에피소드 마감.
- **전환점**: 없음 (단일 감정 지속, 에피소드 엔딩 큐).
- **페이싱**: 정적. 감정적 여운 극대화. F.O으로 잔잔히 마무리.
- **음악 시사**: 서러움·그리움 언더스코어. F.O까지 잔잔히 페이드. 37화 전체 마무리 큐.
