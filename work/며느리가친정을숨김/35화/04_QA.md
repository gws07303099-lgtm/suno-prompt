## 바이블 적합성 최종 확인

- **금기 침범 없음**: 보컬·가사·EDM드롭·포크기타·국악·코믹뽕짝 미사용 ✓
- **라이트모티프**: Q1에 지수 정체 테마(암시), Q3·Q4에 빌런/정숙 테마 배치 ✓
- **다이내믹 설계**: Q1(재벌 위엄 저강도) → Q2(빌드 점증) → Q3(빌런 긴장 조임) → Q4(폭발 스팅어) 4단계 낙차 ✓
- **정숙 폭발 = 빌런 폭발**: 사이다(D타입) 아닌 G타입 스팅어+빌런 테마로 처리, 카타르시스 주지 않음 ✓
- **숏폼 길이 준수**: Q1~Q3 15-25초, Q4 3-5초, 모두 `인트로 즉각 진입` 지시 ✓
