# 작품 음악 바이블 — 「며느리가 친정을 숨김」

> 숏폼드라마(세로형) · 전 50화 · 회당 짧은 단막 호흡 · 전 인스트루멘탈(보컬/가사 없음) · SUNO v5.5 기준
> 이 문서는 화별 파이프라인(스팟팅 디렉터 → 프롬프트 라이터)이 매 화 참조하는 상위 기준이다.

---

## 0. 한 줄 정의 (작업 시작 전 항상 상기)
**"우아한 가면 아래 끓는 분노, 그리고 터질 때의 시원한 파괴."**
재벌가 다이아수저의 절제된 클래스(클래시컬·시네마틱)와 막장 시월드의 신경질적 압박(저음 텐션), 그리고 사이다 응징의 카타르시스(트레일러형 임팩트)를 한 작품 안에서 코드로 묶는다.

---

## 1. 작품 음악 톤 & 전체 컨셉
이 작품은 **"신분 위장 → 굴욕 누적 → 폭발적 응징"** 의 3막 감정 곡선을 50화 내내 반복·증폭하는 사이다 복수극이다. 따라서 음악의 정체성은 **'억눌림과 해방의 낙차'** 그 자체다. 평상시(위장·굴욕 구간)는 음악을 의도적으로 차갑고 절제되게 깔아 관객의 답답함을 음악적으로 공모하고, 반격·정체 공개·응징 순간에 **시네마틱 트레일러/하이브리드 오케스트라**로 폭발시켜 카타르시스를 극대화한다. 전체 사운드의 격(格)은 '재벌가'의 고급스러움을 기준선으로 잡는다 — 싸구려 막장 효과음 톤이 아니라, **모던 시네마틱 + 네오클래시컬**을 베이스로 깔아 작품의 클래스를 음악이 보증한다. 동시에 시월드 빌런 구간에서는 이 고급 질감을 일부러 비틀어(불협·신경질적 스타카토·저역 드론) '겉은 멀쩡하나 속은 썩은' 시댁의 이중성을 사운드로 번역한다.

---

## 2. 장르/스타일 팔레트

### 메인 (작품의 척추)
- **Modern Cinematic / Dramatic Underscore** — 모든 큐의 기본 베드. 시네마틱 스트링 + 피아노 + 미니멀 퍼커션.
- **Hybrid Orchestral / Trailer** — 사이다·반전·응징의 폭발 구간. 오케스트라 + 일렉트로닉 임팩트/브레이즈/사브브레인.
- **Neoclassical Piano** — 재벌가의 우아함, 부부애, 회상, 정체 테마의 서정.

### 서브 (색채·상황별)
- **Tension / Suspense Drone** — 비밀 발각 위기, 위장 유지 긴장.
- **Sinister / Villain Cue** — 시월드 갑질, 빌런 등장. 불협·프리페어드 피아노·저역 펄스.
- **Elegant Ambient / Luxe** — TK그룹, 회장가, 호텔/파티 등 부의 공간감.
- **Quirky Tension (light)** — 시누이·동서의 얄미운 깐족, 코믹 악역 무드(과하지 않게).
- **Minimal Emotional** — 도원 혼수·서러움의 정적, 눈물 구간.

---

## 3. 악기 팔레트

### 코어 악기 (작품 전체를 관통, 일관성의 축)
- **피아노** — 작품의 시그니처 악기. 우아함(클린 그랜드)과 불안(프리페어드/뮤트, 단음 반복)을 동시에 담당. 정체 테마의 중심.
- **시네마틱 스트링** — 솔로 첼로/바이올린(서정·고독) ↔ 풀 스트링 스타카토/오스티나토(긴장·추진).
- **저역 베드** — 신스 베이스 드론, 서브베이스 펄스. 시월드 압박감과 위협의 토대.

### 색채 악기 (정서별로 투입)
- **하프 / 글로켄슈필 / 첼레스타** — 부·우아함·재벌가 광채.
- **타악 임팩트(Braams, Hits, Risers, Sub-drops)** — 사이다·반전·트레일러 빌드.
- **신경질적 퍼커션(staccato strings pizzicato, ticking clock, tense plucks)** — 갑질·깐족.
- **솔로 첼로 / 잉글리시 호른** — 서러움·고독·도원 부재.
- **모던 신스 펄스 / 아르페지에이터** — 전략·계략·복수 설계 장면의 '머리 굴러가는' 질감.

### 피해야 할 것 (금기)
- **보컬·가사·콰이어 멜로디 라인** — 전곡 인스트루멘탈. (텍스처용 무가사 ambient pad/콰이어는 허용하되 멜로디 부르는 보컬 금지. 프롬프트엔 항상 `no vocals` 명시)
- **싸구려 막장 효과음 톤** (과장된 둥둥, 코믹 뽕짝, MIDI 트로트) — 작품 격을 떨어뜨린다.
- **EDM 드롭/페스티벌 신스, 트랩 하이햇 난사** — 톤 이질감. (단, 응징 빌드에 한해 절제된 하이브리드 임팩트는 허용)
- **어쿠스틱 기타 포크/컨트리, 밝은 우쿨렐레** — 작품 무드와 충돌.
- **명백한 시대극/국악 색** (작품은 현대 도시극).

## 4. 감정/기능별 음악 매핑

> 표기: 장르 → BPM → 키 성향 → 핵심 악기 → 에너지 곡선
> SUNO 프롬프트 작성 시 이 항목을 토대로 Style 문구를 구성한다.

### A. 시월드 갑질 · 압박 긴장
- 장르: Sinister Tension / Dramatic Underscore
- BPM: 70–95 (조이는 미드템포) / 키: 마이너, 반음 긴장(트라이톤·불협)
- 악기: 저역 드론 + 프리페어드 피아노 단음 반복 + 스타카토 스트링 + ticking 퍼커션
- 에너지: 낮게 깔린 채 서서히 압박 증가, 해소 없이 컷. **카타르시스 주지 말 것** — 답답함 유지.

### B. 굴욕 · 서러움 (지수가 참는 구간)
- 장르: Minimal Emotional / Neoclassical
- BPM: 50–65 (느림) / 키: 마이너, 단조 서정
- 악기: 솔로 피아노 + 솔로 첼로/잉글리시 호른 + 얇은 스트링 패드
- 에너지: 정적·여백 많게. 멜로디 한 줄로 고독. 절대 화려하게 가지 말 것.

### C. 비밀 / 위장 서스펜스 (정체 들킬 위기)
- 장르: Suspense Drone / Mystery
- BPM: 60–80 (느린 심박) / 키: 모호한 조성, 서스펜디드 코드
- 악기: 저역 드론 + 신스 펄스(심장박동) + 하이 스트링 하모닉스 + 미세 틱
- 에너지: 긴장 빌드 → 아슬아슬 유지. 위기 회피 시 짧은 안도 해소.

### D. 사이다 · 반격 · 응징 (작품의 하이라이트)
- 장르: Hybrid Orchestral / Epic Trailer
- BPM: 90–130 (빌드는 점증, 드롭에서 임팩트) / 키: 마이너 → 파워풀, 단호한 리듬
- 악기: 풀 오케스트라 스트링 오스티나토 + 브라스 스탭 + Braam/Hit/Riser + 서브드롭 + 드라이빙 퍼커션
- 에너지: **명확한 빌드업 → 강한 임팩트(드롭)**. 숏폼 특성상 빌드 짧게, 임팩트 빠르게. 카타르시스 100% 방출.

### E. 재벌가 위압 / 우아함 (TK그룹·정체 공개의 클래스)
- 장르: Elegant Cinematic / Luxe Neoclassical
- BPM: 60–85 / 키: 메이저-마이너 혼합, 당당한 화성
- 악기: 그랜드 피아노 + 풀 스트링(레가토) + 하프/첼레스타 + 절제된 브라스 + 큰 공간 리버브
- 에너지: 위엄 있게 상승, 컨트롤된 파워. '돈의 무게'를 느리고 묵직하게. (정체 공개 순간엔 D와 크로스오버)

### F. 부부애 · 따뜻함 (지수-도원)
- 장르: Warm Neoclassical / Emotional Piano
- BPM: 65–85 / 키: 메이저, 따뜻한 화성
- 악기: 따뜻한 피아노 + 솔로 바이올린/첼로 + 부드러운 스트링 패드 + 미세 글로켄
- 에너지: 잔잔하고 포근. 회상·플래시백에서 톤 살짝 어둡게(상실의 그림자).

### G. 충격 / 반전 (본색 폭로·혼수 등 터닝)
- 장르: Cinematic Shock / Stinger
- BPM: 자유(임팩트 중심) / 키: 불협 → 무조
- 악기: Sudden Braam/Hit + 스트링 글리산도 상승 + 서브드롭 + 정적(silence) 활용
- 에너지: **순간 임팩트 → 급정지 또는 저역 잔향**. 컷 편집과 동기화 전제. 매우 짧게(3–8초 스팅어형도 빈번).

### H. 계략 · 복수 설계 (지수가 머리 굴리는 전략 장면)
- 장르: Modern Tension / Heist-like
- BPM: 95–120 / 키: 마이너, 쿨한 그루브
- 악기: 신스 아르페지오 + 피치카토 스트링 + 미니멀 클릭 퍼커션 + 저역 펄스
- 에너지: 쿨하고 절제된 추진력. '냉철한 지수'의 지능형 무드. 화려한 폭발은 D에서만.

## 5. 반복 음악 모티프 (라이트모티프) 전략

50화 장편이므로 **3개 핵심 테마를 라이트모티프로 운용**, 매 화 변주해 일관성과 누적 카타르시스를 만든다. 멜로디 라인은 단순(3–5음)하게 잡아 변주·이조·악기 교체가 쉽게 한다.

1. **지수 정체 테마 ("다이아수저 테마")**
   - 정체: 우아한 피아노 + 솔로 스트링의 4음 모티프. 평소엔 '위장' 상태라 단조·축소·뮤트로 숨겨 연주.
   - 운용: 정체가 드러나는 순간 → 같은 모티프를 풀 오케스트라 메이저로 폭발 전환(E↔D 크로스오버). 작품 최대 카타르시스 장치. **모티프의 가면을 벗기는** 식의 변주가 핵심.

2. **복수 / 사이다 테마**
   - 정체: 단호한 리듬 + 상승 오스티나토의 추진 모티프. 빌드 → 임팩트 구조 내장.
   - 운용: 매 응징 씬마다 등장하되 타깃별로 강도 상승(도희 < 도진/애라 < 박정숙). 마지막 박정숙('메인 디쉬') 응징에서 풀버전 클라이맥스.

3. **빌런(시월드) 테마 — 특히 박정숙**
   - 정체: 프리페어드 피아노 단음 + 불협 저역 + ticking. '겉은 점잖은 척, 속은 짐승'의 이중 질감.
   - 운용: 시댁 갑질마다 깔리며 관객의 분노를 적립. 박정숙은 묵직한 저역 버전, 도희는 신경질적 피치카토 경량 버전, 도진/애라는 quirky tension 변주로 분기.

> 라이트모티프 운용 원칙: **모티프는 멜로디·리듬 핵만 공유하고 악기·조성·템포는 정서에 맞춰 자유 변주**. 매 화 동일 음원 재탕이 아니라 '같은 DNA의 다른 옷'으로 뽑는다. 프롬프트 라이터는 해당 씬이 어떤 테마 계열인지 먼저 판단 후 4번 매핑으로 색을 입힌다.

## 6. 숏폼 큐 운용 원칙 (세로형 단막 대응)

- **길이**: 큐당 15–45초가 표준. 스팅어/반전은 3–10초. SUNO 생성분에서 필요한 구간만 트림 전제로 프롬프트는 '핵심 무드 즉시 진입'을 지시한다.
- **인트로/훅 타이밍**: 숏폼은 빌드 길게 못 끈다. **인트로 0–3초 내 무드 확정**. 프롬프트에 `immediate mood, short intro, no long buildup` 류 지시 포함. 사이다 큐도 빌드 4–8초 내 임팩트 도달.
- **반복성**: 한 큐 안에서 루프 가능한 구조(8–16마디 단위)로 뽑아 편집 신축성 확보.
- **컷 동기화**: 반전/응징은 영상 컷과 임팩트 일치가 생명 → 스팅어·드롭형은 '명확한 타격점 1회'를 강조해 생성.
- **다이내믹 낙차 설계**: 같은 화 안에서 굴욕(B/저에너지) → 응징(D/고에너지)의 낙차를 의도적으로 크게. 작품의 사이다는 음악 다이내믹 대비에서 나온다.
- **격 유지**: 어떤 큐든 베이스 질감은 '시네마틱/네오클래시컬'. 금기 톤(싸구려 막장 효과음) 침범 금지 — 이게 작품의 클래스를 지키는 마지노선.
- **SUNO 프롬프트 공통 꼬리**: 모든 큐 Style 끝에 `instrumental, no vocals` 명시. Style ~1000자 한도 내 작성, 별도 Exclude 필드 없음(인라인 네거티브로 처리).
