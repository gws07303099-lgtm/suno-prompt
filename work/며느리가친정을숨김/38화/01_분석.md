## 씬 단위 분해

### 블록 A — 대저택 입장 · 감탄 (대본 1~11행)

- **장소/시간**: 저택 거실, 입장 직후
- **등장인물**: 정숙, 도희
- **행동**: 인테리어·대리석 바닥 감탄, 지수 사진 아직 보기 전
- **감정선**: 얼떨떨한 놀라움 + 코믹한 촌스러움 (파리 미끄러져 전치 3주)
- **갈등**: 없음 — 공간이 주인공
- **페이싱**: 가볍고 경쾌, 코믹 리듬

### 블록 B — 빌런 의도 선언 · 이혼 작전 (대본 12~20행)

- **장소**: 동일
- **행동**: 도원 부재 확인 → 정숙의 이혼·위자료 작전 선언 → 지수 사진액자 발견 후 떨떠름
- **감정선**: 정숙의 적대적 전략 발화, 도희 살짝 저항("그럼 안될텐데")
- **갈등**: 정숙(이혼 강행) vs 도희(미온적 동조)
- **전환점**: 지수 사진액자를 보는 순간 — 정숙의 표정 전환 (소름, 부르르)
- **페이싱**: 긴장감 서서히 올라가며 이분법적 압박

### 블록 C — 도희 추궁 · 등짝 후려침 (대본 26~40행)

- **장소**: 동일
- **행동**: 도희가 지수 편들기 → 정숙의 의심·추궁 → 도희 고백(이미 알고 있었음) → 정숙 분노, 등짝 후려침
- **감정선**: 정숙의 의심→분노 폭발, 도희의 당혹·항변
- **갈등**: 정숙 vs 도희 — 정보 은폐에 대한 배신감
- **페이싱**: 빠른 대화 교환, Quirky 신경질적 긴장, 등짝 타격 임팩트 포인트
- **음악 기능**: 코믹 빌런 압박 — 무겁지 않되 신경질적

### 블록 D — TK그룹 정체 폭로 · 정숙 멘붕 (대본 41~50행)

- **장소**: 동일
- **행동**: 도희가 "TK그룹 윤성철 회장!" 외침 → 정숙 혼란, 어안이 벙벙
- **감정선**: 도희 폭로(답답함 폭발) → 정숙의 충격·사고 정지
- **갈등**: 없음 — 정보의 충격이 모든 갈등을 삼킴
- **전환점**: 이 화의 클라이맥스 & 38화 전체 반전. 지수 정체 테마 첫 폭발적 암시 지점.
- **페이싱**: 폭발 후 급정지 — 정적과 여운

## 감정 곡선 (전체)

```
[입장 감탄] ─ 가볍고 코믹 ──→ [이혼 작전 선언] ─ 빌런 압박 ──→ [추궁/후려침] ─ 신경질 긴장 ──→ [TK 폭로] ─ 충격 폭발 → 여운
낮음~중간                      중간~높음                          중간(Quirky)                  최고점→정적
```

## 음악 설계 키포인트

1. **저택 입장**: 재벌가 클래스의 위압감이 음악에 먼저 등장해야 관객도 공간 규모를 느낀다. Elegant Luxe 톤 + 살짝 Quirky (촌스러운 방문객의 코믹 반응).
2. **빌런 선언**: 바이블 A타입 (시월드 갑질) + 빌런 테마 (박정숙 버전). 카타르시스 주지 않고 답답함 적립.
3. **추궁·후려침**: 바이블 Quirky Tension 타입 (도희 버전 경량). 무거운 악당 테마가 아니라 신경질적 피치카토 계열.
4. **TK 폭로**: 바이블 G타입(충격/반전) + E타입(정체 테마 발아) 크로스오버. 지수 '다이아수저 테마' 라이트모티프 첫 과감한 암시.
