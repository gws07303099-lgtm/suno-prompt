- **C**: 48 BPM / C minor / solo piano dissonance only / Styles ~175자 ✅

- **A**: 58 BPM / A minor / piano + violin / Styles ~185자 ✅

- **C**: 55 BPM / G minor / celesta + violin / Styles ~180자 ✅

### EP30_Q4 (위협예감·긴장빌드·정숙테마재진입)

- **B**: 68 BPM / C minor / cello drone + spiccato strings + ticking / Styles ~215자 ✅
- **C**: 64 BPM / B minor / sub drone + dark ambient pad + orchestral hit / Styles ~220자 ✅
- 3변형 음악적 차별성: A=신스전자 드론 / B=어쿠스틱 현악 / C=앰비언트 질감 → 서로 다른 음색 계통 ✅

- SFX 컷 타이밍 메모 기재 ✅

### EP30_Q5 (클리프행어·빌런등장충격)

- 3변형 음악적 차별성: A=브람+드롭 표준 / B=풀오케스트라 웅장 / C=피아노클러스터 날카로움 ✅
