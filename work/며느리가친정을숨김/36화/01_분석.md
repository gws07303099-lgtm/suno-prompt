# 36화 대본 분석

## 씬 구조 개요
36화는 **씬 #68** 단일 씬으로 구성. 고급 식당 룸이라는 제한된 공간에서 정숙의 행패 → 경찰 사진 폭로 → 지수의 배경 공개 결심 → 도원의 냉정한 이탈까지 압축된 단막 전개.

## 씬 #68 — 고급 식당 룸 안 / 낮

**장소**: 고급 식당 VIP 룸  
**시간**: 낮  
**등장인물**: 정숙(시어머니·빌런), 도원(남편), 지수(주인공), 지수 엄마, 윤회장(지수 부친)

### 갈등 구조
- **메인 갈등**: 정숙이 윤회장 부부 앞에서 사기꾼 몰이 행패 → 지수가 맞서며 부모에게 상견례 제안
- **서브 갈등**: 정숙이 경찰 연행 사진을 공개해 도원의 신뢰를 흔들려는 시도
- **전략 갈등**: 지수가 배경 숨기기를 포기하고 정체 공개를 결심하는 역공 전환
- **충격 엔딩**: 도원이 오히려 지수를 두고 어머니와 퇴원하겠다는 냉정한 선언

### 감정선 (시간 순)
1. **분노·압박** — 정숙 행패 개시: "세상 천지에 어떤 사람이 VIP 병실비에 5천만원짜리 산삼을 턱턱 내줘?!" → 윤회장 부부 어이없음, 도원 말림
2. **고조·적대** — 정숙의 사기꾼 공격 + 지수의 반박: "모든 사람이 그런 쓰레기 같은 맘을 갖고 살진 않아요" → 정숙 양주먹 꽉 쥠, 도원 흠칫
3. **위기·의심** — 경찰 연행 사진 공개: 도원 "이거 진짜.. 지수.. 씨예요?" → 도원에 의심 씨앗 심기
4. **전략적 전환** — 지수 부모 상견례 초대, 지수 nar "그래.. 이렇게 된 마당에 더 이상 숨길 것도 없겠지" → 지수의 배경 공개 결심
5. **경멸·조롱** — 정숙의 냉소: "냄새가 난다, 냄새가 나~ 사기꾼 냄새!!" → 지수 내면 "직접 확인해봐"로 당당하게 맞섬
6. **충격·절망** — 도원의 예상치 못한 선언: "아무래도 안되겠습니다. 이만 퇴원할게요" → 지수 nar "안돼.."

### 전환점
- **1차 전환점**: 경찰 사진 공개 — 도원의 신뢰가 흔들리는 위기 (의심 고조)
- **2차 전환점**: 지수 nar "이렇게 된 마당에" — 숨기기에서 공개로 지수의 전략 전환 (역공 결심)
- **3차 전환점(엔딩 충격)**: 도원의 "이만 퇴원할게요" — 지지군이 돼야 할 도원이 오히려 지수를 버리는 반전 (다음 화 빌드업)

### 페이싱
단일 씬의 압축 전개. 정숙의 고성으로 고에너지 시작 → 경찰 사진 폭로로 긴장 폭발(스팅어 포인트) → 상견례 합의로 잠시 숨 고르기 → 정숙 조롱으로 다시 빌런 기운 올라옴 → 지수의 결연한 내면 → 도원의 냉정한 선언으로 충격 엔딩. 화의 마지막이 "안돼.."라는 절망으로 닫혀 다음 화로 긴장이 그대로 넘어간다.

### 음악 스팟팅 메모
- **큐 없이 침묵이 강한 구간**: 경찰 사진 스팅어(Q2) 이후 ~ 지수 nar 시작(Q3) 전까지 — 지수 부모의 온화한 초대·정숙의 반응 대화는 정적 속 대사만으로 긴장 유지.
- **Q1→Q2 직결**: 빌런 언더스코어가 압박 최고조에서 스팅어로 이어짐.
- **Q3→Q4 의도적 침묵 간격**: 지수 nar "직접 확인해봐" 후 도원이 앞에 서기까지 짧은 정적 배치.
