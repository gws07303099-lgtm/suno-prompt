# 33화 QA 검수

> 가이드 §7 체크리스트 기준. 큐시트(02)·프롬프트(03) 원문을 직접 대조 검수.  
> 총 4개 큐 × 3개 변형 = 12개 프롬프트.

---

## 검수 체크리스트 (§7)

- [x] Styles 글자수 ≤1000자 (권장 ≤950)
- [x] `Lyrics 탭: Write` 명시 + Write 블록 가사 단어 0
- [x] Styles 끝 `instrumental, no vocals` 유지
- [x] BPM·키·장르·악기 2개+ 포함 (스팅어 BPM 면제)
- [x] 모순 키워드 없음 / 언어 혼입·오타 없음
- [x] 아티스트 실명 미사용
- [x] CUE_ID 큐시트(02)와 1:1 일치
- [x] 작품 내 톤/악기 팔레트 일관성 (바이블 준수)
- [x] 변형 3개(A/B/C) 존재, 단순 rewording 아닌 음악적 구별
- [x] more options(Exclude Styles / Weirdness / Style Influence) 미기재
