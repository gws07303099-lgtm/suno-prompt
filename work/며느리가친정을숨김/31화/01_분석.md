## 씬 #62 — TK병원 VIP병실 · 밤

| 항목 | 내용 |
|---|---|
| 장소/시간 | TK병원 VIP병실 · 밤 |
| 등장인물 | 도원, 지수, 의사 |
| 갈등 | 혼수에서 막 깨어난 도원이 위험을 무릅쓰고 병실을 나가려 함 → 지수와 의사가 만류 |
| 감정선 | 긴장·불안(지수), 혼란·충동(도원), 긴박(의사) |
| 전환점 | 도원이 두 사람을 뿌리치고 복도로 나감 |
| 페이싱 | 빠름 — 짧은 대화, 즉각적 행동 |

**감정 메모**: 도원은 기억을 잃은 채 깨어난 상태로, 지수가 누구인지조차 모른다. 지수는 아직 상황을 설명하지 못한 불안과 간절함이 공존하는 복잡한 감정. 의사는 의학적 긴박감만을 대변하는 중립 캐릭터.

## 씬 #63 — TK병원 VIP병실 앞(복도) · 밤

| 항목 | 내용 |
|---|---|
| 장소/시간 | TK병원 VIP병실 앞 복도 · 밤 |
| 등장인물 | 도원, 정숙, 가드, 지수, 의사, 간호사 |
| 핵심 갈등 | 도원의 기억상실 상태에서 정숙(시어머니)과 지수(아내)가 서로 상대방을 '살인 미수 가해자'로 지목하며 정면충돌 |

### 씬 #63 내부 흐름 (박자 단위)

| # | 박자 | 내용 | 감정/기능 |
|---|---|---|---|
| 63-a | 도원 등장 | 정숙이 "아들!!" 소리침. 가드 가로막음. 도원이 가드를 성큼성큼 밀침 | 물리적 충돌 개시, 긴장 폭발 시작 |
| 63-b | 주 보호자 충격 | 지수가 "저에요, 제가 요청했어요" 밝힘 → 도원 "당신이? 무슨 자격으로" | 반전 / 권력 역전 — 기대와 다른 구도 |
| 63-c | 정숙 폭로 | "이년이 너 누워있을 때, 보험금 노리고 너 죽이려고 한 년이란다" | 빌런 공격 / 갈등 급등 |
| 63-d | 지수 반박 | "당신을 죽이려고 한건 어머니랑 가족들이야" | 사이다 예비 / 진실 폭로 맞대결 |
| 63-e | 도원 혼란·발작 | "아..아악!" 머리 감싸쥐고 쪼그려 앉음 | 드라마틱 클라이맥스 / 충격 정점 |
| 63-f | 의사 개입 | "기억 회복 영구 방해 가능" 경고 | 긴장 일시 멈춤 / 의학적 경고 |
| 63-g | 기억상실 공식화 | 도원: "기억이 하나도 안 나요" / 간호사: 기억상실 증세 공식 언급 | 충격 후 정적 / 상황 재정의 |
| 63-h | 빌런 씰룩 | 정숙이 기억상실 사실 인지 후 입꼬리를 씰룩대는 것으로 마무리 | 새로운 음모 예고 / 빌런 테마 엔딩 |

**감정 메모**:
- 정숙: 공격자(자기가 피해자인 척 위장). 기억상실 사실을 알자마자 새로운 계산이 시작되는 표정으로 마무리. 31화 내내 '겉은 피해자, 속은 가해자'의 이중성 극대화.
- 지수: 간절하나 무력함. 도원이 믿지 않는 상황에서 진실을 말하지만 공허하게 들림. 주 보호자 설정은 유일하게 지수가 쥔 카드.
- 도원: 혼란의 희생자. 기억상실로 인해 판단 불가 상태. 두 여자의 상충하는 주장 앞에서 육체적 발작으로 무너짐.
