핵심 기능
🎬
대본 · 큐 동기화 뷰
대본 원문 위에 큐 범위를 색 띠(band)로 오버레이. 큐 ID 라벨 클릭 시 해당 큐 카드로 자동 스크롤. 큐시트 앵커(IN/OUT 텍스트)를 파싱해 대본 위치를 결정적으로 매핑.
v0.1~
🤖
Claude 헤들리스 자동 스팟팅
claude -p 를 서버에서 subprocess로 실행. 구독 인증 재사용(API 키 불필요). 잡 레지스트리로 진행 상태 폴링. 완료 시 큐 카드 자동 갱신.
v0.1~
🎵
A / B / C 3변형 탭
큐마다 음악적으로 다른 세 가지 해석(A=정공법, B=대안 색, C=실험적)을 탭으로 비교. Write 블록·Styles 블록 개별 복사 버튼. 선택(Select) · 킵(Keep) 배지 저장.
v0.3~
🖱️
드래그 구간 오버라이드
대본 위 큐 색 띠 양 끝의 핸들을 드래그해 큐 커버 구간 재조정. char offset을 selection.json에 저장. 재생성 시 Claude에게 해당 발췌 구간 우선 주입.
v0.4~
💬
감독 코멘트 → 재생성 반영
큐 카드 코멘트 영역에 수정 지시·분위기 메모를 입력하고 재생성 버튼을 누르면, 코멘트가 Claude 프롬프트의 note로 주입되어 반영.
v0.4~
⚡
single_cue 빠른 재생성
큐 하나만 재생성할 때 full 파이프라인 대신 single_cue 모드로 실행. 모델을 sonnet → haiku로 전환해 속도 약 50~70% 단축. A·B·C 세 변형 모두 교체.

cueparse.py
대본 파싱 + 앵커 매핑 + selection.json 읽기/쓰기.
