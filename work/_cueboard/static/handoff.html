다음 작업
지금 바로 할 일
39화부터 순서대로 스팟팅합니다. 한 화씩 진행하세요.
1

서버 실행

터미널에서 python work/_cueboard/server.py 실행 → 브라우저에서 localhost:8765 접속

2

화 폴더 준비

work/며느리가친정을숨김/39화/ 폴더 생성 → 대본 텍스트를 00_대본raw.txt로 저장 (파일명 정확히)

3

자동 스팟팅 실행

Cue Board 좌측 화 목록에서 39화 선택 → "자동 스팟팅" 버튼 클릭 → 로그창 확인하며 약 2~3분 대기. SPOT_DONE 메시지가 뜨면 완료.

4

결과 검토 및 선택

큐 카드의 A·B·C 탭을 전환하며 비교 → 마음에 드는 변형에 ★ 선택 표시. 수정이 필요하면 코멘트 입력 후 "재생성".

5

커밋 (작업 저장)

한 화 완료 후 git commit. 제목 예시: 39화 스팟팅 데이터 (자동 생성) — 데이터 커밋은 버전 태그 없음.

주의사항
꼭 확인하세요
대본 파일명 규칙 준수 — 반드시 00_대본raw.txt 로 저장. 파일명이 다르면 자동 스팟팅이 대본을 찾지 못함.
selection.json은 git에 올리지 않음 — 선택·킵·코멘트 상태는 로컬 파일로만 관리. 커밋 시 포함 금지.
서버 실행 필수 — Cue Board는 로컬 웹앱. python work/_cueboard/server.py 로 서버 켜야 동작. 브라우저 단독 실행 불가.
Claude API 키 필요 — 자동 스팟팅·재생성은 Claude API를 사용. ANTHROPIC_API_KEY 환경변수가 설정되어 있어야 함.
