# 음악 바이블 — 「엄마의 남자를 뺏겠습니다」 (밤부네트워크)

> 작성: music-project-planner 단계 산출물 / SUNO v5.5 인스트루멘탈 기준 / 전 큐 가사 없음

## 1. 작품 개요
- 포맷: 숏폼드라마, 전 50화 (화당 1~2씬, 약 1~2분)
- 장르: **치정 × 복수 × 로맨스 스릴러 (막장)** — #치정 #복수 #자극 #로맨스 #새엄마 #삼각관계
- 로그라인: 아빠가 죽고 새엄마 **유화경**에게 모든 걸 빼앗긴 **민도아(23)**가, 화경이 사랑하는 남자 **강무영(28)**을 빼앗아 복수하려다 진짜 감정에 휘말린다.
- 핵심 인물별 음악 함의:
  - **민도아(주)** — 상실 → 복수 결의 → 흔들리는 감정. 다크하지만 인간적.
  - **유화경(메인 빌런/새엄마)** — 우아한 가면 + 차가운 위협. 빌런 테마.
  - **강무영(남주/타깃이자 사랑)** — 금기의 멜로. 따뜻하지만 위태로움.

## 2. 전체 음악 톤
- 한 단어: **"우아하게 위험한"**. 고급스러운 시네마틱 + 막장 특유의 강렬·자극.
- 다크·텐션 베이스 위에 멜로(따뜻함)와 스팅어(충격)가 교차. 숏폼이라 **빠른 후킹 + 짧고 강한 큐**.

## 3. 악기 팔레트 (코어 사운드)
- **현(strings)**: 스타카토(긴장)·레가토(감정)·트레몰로(불안)
- **피아노**: 다크 그랜드(서스펜스)·펠트/프리페어드(친밀·회상)
- **신스**: 로우 펄싱 베이스, 다크 드론, 서브 부밍(충격)
- **타악**: 틱킹/펄스(긴장), 오케스트럴 힛·브라스 스탭(스팅어)
- **질감**: 와이드 시네마틱 스테레오, 리버브 깊은 회상 톤

## 4. 테마/모티프 전략 (반복 사용)
- **복수 테마(도아)**: 단조, 결연한 저음 오스티나토 + 상승 스트링. 도아가 결심·행동할 때 변주.
- **빌런 테마(화경)**: 우아한 왈츠풍/하프시코드 느낌 + 불협 긴장. 겉은 우아, 속은 차가움.
- **금기 멜로(도아×무영)**: 따뜻한 펠트 피아노 + 라이트 스트링, 끝맛에 불안한 텐션 한 방울.
- 같은 감정 = 같은 악기/조성 팔레트로 회차 간 일관성 유지.

## 5. 금기 (쓰지 말 것)
- 가사/보컬 일절 없음 (전 큐 instrumental only).
- 장르 톤을 깨는 과한 코믹·밝은 팝. 코믹 비트는 절제해서만.
- 아티스트 실명 → 질감 묘사로 우회.

## 6. 큐 우선순위 정책
1순위: 충격 오프닝/반전 **스팅어** (후킹의 핵심)
2순위: 감정 절정 (상실·배신·눈물)
3순위: 긴장 빌드 (서스펜스·대치)
4순위: 멜로/로맨스
의도적 **침묵**도 연출 수단 — 과한 스코어링 금지.
